# NoorJs Project Creator Command

## Installation

To create new NoorJs app run

**NPM**

```bash
npx create noor@latest
```

**Yarn**

```bash
yarn create noor
```

**PNPM**

```bash
pnpm create noor
```

**Bun**

```bash
bun create noor
```

## Templates

- js (JavaScript)
- ts (Typescript)
- tailwind-js (tailwindCss + JavaScript)
- tailwind-ts (tailwindCss + TypeScript)
